<p>You can make things <strong>bold</strong> or <em>italic</em> or <strong><em>both</em></strong> or <em><strong>both</strong></em>.</p>
<p>Character formatting extends across line breaks: **bold,
this is still bold. This line deliberately does not end in star-star.</p>
<p>Not bold. Character formatting does not cross paragraph boundaries.</p>
<p>You can use <a href="http://www.example.org/wiki/internal-links">internal links</a> or <a href="http://www.wikicreole.org" class="external">external links</a>,
give the link a <a href="http://www.example.org/wiki/internal-links">different</a> name.</p>
<p>Here&#039;s another sentence: This wisdom is taken from <a href="http://www.example.org/wiki/Ward_Cunninghams" class="external">Ward Cunningham&#039;s</a>
<a href="http://www.c2.com/doc/wikisym/WikiSym2006.pdf" class="external">Presentation at the Wikisym 06</a>.</p>
<p>Here&#039;s a external link without a description: <a href="http://www.wikicreole.org" class="external">http://www.wikicreole.org</a></p>
<p>Be careful that italic links are rendered properly:  <em><a href="http://my.book.example/" class="external">My Book Title</a></em></p>
<p>Free links without braces should be rendered as well, like <a href="http://www.wikicreole.org/" class="external">http://www.wikicreole.org/</a> and <a href="http://www.wikicreole.org/users/~example" class="external">http://www.wikicreole.org/users/~example</a>.</p>
<p>Creole1.0 specifies that <a href="http://bar" class="external">http://bar</a> and <a href="ftp://bar" class="external">ftp://bar</a> should not render italic,
something like foo://bar should render as italic.</p>
